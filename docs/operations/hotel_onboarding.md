# ホテルスタッフ 利用開始ガイド

**対象:** ご担当のホテルスタッフの方  
**所要時間:** 約10分  
**サポート:** 東京エムケイ 配車係（下記連絡先）

---

## 1. はじめに

シャトルハイヤー予約システムは、ホテルご宿泊のお客様の成田空港送迎予約を  
フロントから素早く完了できる専用システムです。

**できること**
- 空港シャトル便の空席確認（リアルタイム）
- 宿泊者の予約登録（30秒で完了）
- QRコードチケットの印刷・案内
- 予約履歴の確認・キャンセル

---

## 2. ログイン

1. ブラウザで `https://shuttle-hire-platform.vercel.app/hotel/login` を開く
2. 東京MKから受け取った **メールアドレス** と **初期パスワード** を入力
3. 「ログイン」を押す

> **初回ログイン後はパスワードを変更してください。**  
> 右上のアカウントメニュー → 「パスワード変更」

---

## 3. 予約の入れ方（基本フロー）

### Step 1：カレンダーで便を選ぶ

ログイン後、カレンダー画面が表示されます。

| 表示 | 意味 |
|---|---|
| 緑色（残席あり） | 予約可能 |
| 黄色（残席わずか） | 残席1〜2 |
| 赤色（満席） | 予約不可 |
| グレー（締切済） | 受付終了 |

乗りたい便のカードをクリックしてください。

### Step 2：お客様情報を入力

| 項目 | 内容 |
|---|---|
| お客様氏名 | パスポートのアルファベット表記（例: YAMADA TARO） |
| 人数 | 乗車人数（大人・子供の合計） |
| フライト番号 | 例: NH001、EK317 |
| 荷物数 | スーツケース等の預け荷物の数 |
| 備考 | 特記事項があれば（車いす対応など） |

フライト番号を入力すると航空会社・ターミナル・出発時刻が自動で表示されます。

### Step 3：QRコードを案内する

「予約する」を押すと **QRコード画面** が表示されます。

- **印刷する場合:** 「印刷」ボタン → チェックアウト時にお渡しください
- **スマホで案内する場合:** QRコードをお客様のスマホで読み取ってもらうだけでOK

QRコードの画面URLはお客様のスマホに送ることもできます（URLをコピーして送信）。

---

## 4. キャンセルの仕方

### ホテルスタッフによるキャンセル

1. 「予約履歴」→ 該当予約を選択
2. 「キャンセル」ボタン → 確認ダイアログで「キャンセルする」
3. 自動でお客様にキャンセル完了メールが届きます

> **出発3時間前以降のキャンセルは東京MKにご連絡ください。**

### お客様自身によるセルフキャンセル

お客様がQRコードページから自分でキャンセルすることも可能です。  
（登録メールアドレス宛にOTPコードが届き、入力して認証）

## 5. よくある質問

**Q: フライト番号を入れても航空会社が表示されない**  
A: 入力してから1〜2秒待ってください。表示されない場合は手入力で進んでいただけます。航空会社情報は参考表示のため、必須ではありません。

**Q: 便が満席で予約できない**  
A: 東京MKの配車係にご連絡ください。増便・代替便を手配いたします。

**Q: 予約したのにQRが表示されない**  
A: ブラウザの「戻る」を押さず、再度「予約履歴」から確認してください。

**Q: お客様から「QRコードが使えない」と言われた**  
A: QRコードのURLをコピーして、お客様のメールやLINEに送ってください。URL自体がチケット代わりになります。

**Q: 誤って予約してしまった**  
A: 出発前日17時までであれば「予約履歴」からキャンセルできます。それ以降は東京MKへご連絡ください。
